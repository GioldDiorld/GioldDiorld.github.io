hexo介绍

hexo是一个博客框架，应用广泛。

hexo前置安装

hexo前置安装：git与node.js

判断node.js是否成功安装：cmd中输入npm -v

判断gt是否成功安装：git bash输入git –version

hexo安装

必须在空文件夹中进行

安装 hexo 框架

npm install -g hexo-cli

初始化文件夹

hexo init

安装 hexo 依赖包

npm install

命令很好理解，第一行安装 hexo 模块，-g表示安装全局模块；第二行是 hexo 初始化，会用 git clone命令去 GitHub 下载一个 hexo 默认模板代码库；第三行是安装依赖包，类似安装 pip 的 requirement 文件，会根据刚下载的代码库中的配置文件，下载并安装所需依赖包。

hexo使用

生成静态网站

hexo g

启动服务器

hexo s

或者直接执行hexo g && hexo s，其中hexo g是 hexo generate命令的简写，hexo s是 hexo server命令的简写。

hexo主题

https://hexo.io/themes/ 用于寻找主题，找到的主题下载到themes文件夹。

之后将根目录下的_config.yml中的theme值改为想要的主题名字。

每个主题都有对应的设置方法，需要在主题文档中寻找

hexo个性化设置

https://hexo.io/zh-cn/docs/ 用以修改网站的各种值，如网站名网站作者等。

git bash中ssh -T git@github.com 命令可以确定是否成功连接上了github。
